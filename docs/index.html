PBL SO-101 Kitchen Robot

Git-downloadable Windows project for SO-101 LeRobot cooking workflows: local teleop, calibrated pose saving, cup pickup, spoon/stick pickup, stirring, dashboard control, ChatGPT brain, and Hugging Face upload/download.

Install Configure Motor IDs Teleop Save Poses Run Record/Train ChatGPT Hugging Face Git

Safety

Keep one hand near the robot power switch during every real movement. Run dry-runs before movement. Do not run pose-saving commands in a batch because every pose must be saved after physically moving the robot to that pose.

1. Download And Install

Fresh Windows System

cd C:\Users\PBL
git clone https://github.com/lakshveerrao/kichen-robot.git so101_kitchen_stirrer_milestone
cd C:\Users\PBL\so101_kitchen_stirrer_milestone
py -3.11 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install uv
uv pip install -e .
pbl --help

If The Folder Already Exists

cd C:\Users\PBL\so101_kitchen_stirrer_milestone
git pull
.venv\Scripts\activate
uv pip install -e .
pbl --help

2. Configure And Check Robot

pbl setup --port COM7 --cameras 1 2
pbl status
mode

Do not use commas for cameras. Use spaces: --cameras 1 2.

With Leader Arm For Teleop

pbl setup --port COM7 --leader-port COM8 --leader-id 1 --follower-id kitchen_stirrer_follower --cameras 1 2

3. Assign Motor IDs And Calibrate

Only assign motor IDs for a new robot or after replacing motors. LeRobot will ask you to connect one motor at a time.

pbl setup-motors --device all
pbl robo --setup-motors all

Calibrate follower only:

pbl calibrate --device follower --follower-port COM7

Calibrate leader and follower:

pbl calibrate --device all --leader-port COM8 --follower-port COM7
pbl robo --calibrate all --leader-port COM8 --follower-port COM7

4. Teleoperation Without Solo CLI

This project does not require Solo CLI. Teleop uses local LeRobot leader/follower APIs.

pbl teleop

Explicit ports:

pbl teleop --leader-port COM8 --follower-port COM7 --leader-id 1 --follower-id kitchen_stirrer_follower

Stop teleop with Ctrl+C.

5. Save Poses

Move the robot to the pose using teleop or by hand, then save only that pose.

Main Poses

pbl save-pose HOME
pbl save-pose ABOVE_BOWL
pbl save-pose STIR_DEPTH
pbl save-pose LIFT
pbl save-pose PARK
pbl save-pose STIR_FORWARD
pbl save-pose STIR_BACK
pbl save-pose STIR_LEFT
pbl save-pose STIR_RIGHT
pbl save-pose UP
pbl save-pose DOWN
pbl save-pose LEFT
pbl save-pose RIGHT

Ingredient Cup Poses

pbl save-pose CUP_APPROACH
pbl save-pose CUP_GRASP
pbl save-pose CUP_LIFT
pbl save-pose CUP_POUR
pbl save-pose CUP_PLACE
pbl save-pose CUP_RELEASE

Spoon/Stick Poses

pbl save-pose STIRRER_APPROACH
pbl save-pose STIRRER_GRASP
pbl save-pose STIRRER_LIFT
pbl save-pose STIRRER_READY

6. Validate And Run

No Movement 
pbl dry-run
pbl dry-run --ingredients
Small Movement Tests 
pbl stir --motion front-back --cycles 1
pbl stir --motion left-right --cycles 1
pbl ingredients --action all
Full Run 
pbl run --ingredients
Smart Tight Grip 
pbl smart --cycles 5 --tight -5 --open 30

Camera Check Override

pbl run --ingredients --skip-camera-check

Dashboard

pbl dashboard --allow-movement

Open http://127.0.0.1:8000.

7. Local Record, Train, Inference

This is a simple local pose-sequence workflow. It does not require Solo CLI.

pbl robo --record --seconds 20 --fps 10
pbl robo --train --input recordings/latest/observations.jsonl --model-out models/latest_policy.json
pbl robo --inference --policy models/latest_policy.json --speed-scale 0.02
pbl robo --replay

8. ChatGPT Brain

Do not put API keys in code or Git.

Command Prompt

set OPENAI_API_KEY=your_new_api_key_here
pbl brain "pick the cup, grab the stick tight, and stir slowly"
pbl brain "pick the cup, grab the stick tight, and stir slowly" --execute

PowerShell

$env:OPENAI_API_KEY="your_new_api_key_here"
pbl brain "pick the cup, grab the stick tight, and stir slowly"
pbl brain "pick the cup, grab the stick tight, and stir slowly" --execute
