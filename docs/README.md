# PBL SO-101 Kitchen Robot Site

This folder contains the GitHub Pages documentation site.

To enable it:

1. Open the GitHub repository.
2. Go to `Settings`.
3. Go to `Pages`.
4. Choose `Deploy from a branch`.
5. Choose branch `main`.
6. Choose folder `/docs`.
7. Save.

Expected URL:

```text
https://lakshveerrao.github.io/kichen-robot/
```
